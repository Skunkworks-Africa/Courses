    1. Adjust Logging Parameters: Modify the number of primary and secondary log files.
      db2 update db cfg for MUSICDB using logprimary 5 logsecond 10

      If prompted, restart the database to apply the changes:

      db2stop force
      db2start
    2. Verify Configuration: Check the configuration settings to ensure they have been applied correctly.
      db2 get db cfg for MUSICDB
    Task 4: Create a Storage Group
    1. Create the Storage Group: Define a new storage group named APP_DATA.
      db2 "CREATE STOGROUP APP_DATA ON '/dbauto/path2'"
    2. Verify Storage Group Creation: Ensure the storage group has been created successfully.
      db2 list stogroups
    Task 5: Create a Tablespace
    1. Create the Tablespace: Define a new tablespace named TSP01 within the APP_DATA storage group.
      db2 "CREATE TABLESPACE TSP01 MANAGED BY AUTOMATIC STORAGE USING STOGROUP APP_DATA EXTENTSIZE 4"
    2. Verify Tablespace Creation: Ensure the tablespace has been created successfully.
      db2 list tablespaces
    Task 6: Execute SQL Scripts
    1. Navigate to the SQL Script Directory: Change to the directory containing your SQL scripts.
      cd $HOME/ddl
    2. Execute the SQL Script: Run the SQL script to create multiple tablespaces.
      db2 -tvf create_tablespaces.ddl
    3. Verify Tablespace Creation: Ensure all tablespaces have been created successfully.
      db2 list tablespaces show detail
    Task 7: Retrieve Tablespace Information
    1. Execute the SQL Script: Run the SQL script to retrieve tablespace information.
      db2 -tvf select_tablespaces.sql
    2. Review the Output: Check the output for detailed tablespace information.
      TBSPACE DEFINER TBSPACEID TBSPACETYPE DATATYPE SGNAME
      SYSCATSPACE SYSIBM 0 D A IBMSTOGROUP
      TEMPSPACE1 SYSIBM 1 S T IBMSTOGROUP
      USERSPACE1 SYSIBM 2 D L IBMSTOGROUP
      SYSTOOLSPACE SYSTEM 3 D L IBMSTOGROUP
    3. Execute Additional SQL Scripts: Run the script to get container-level details.
      db2 -tvf select_mon_get_cont.sql
    4. Execute Disk Path Information Script: Run the script to retrieve disk path information.
      db2 -tvf dbpaths.sql
    5. Review Disk Path Information: Check the output for disk path details.
